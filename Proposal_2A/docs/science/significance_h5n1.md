# Why H5N1 Is a Strong Scientific Use Case

H5N1 is scientifically compelling because it combines:
- rapid sequence diversification
- high concern around host adaptation
- strong relevance of HA and related proteins to structure-function questions
- a meaningful need to connect mutation, structure, antigenicity, and function

For this proposal, H5N1 should be presented primarily as a demanding biomolecular modeling problem rather than a general surveillance issue.
