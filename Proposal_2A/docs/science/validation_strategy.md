# Validation Strategy

## Principles
- define tasks before selecting favorite models
- include sequence-only and conventional baselines
- evaluate both prediction and interpretability
- test generalization, not just random splits
- report uncertainty where possible

## Recommended evaluation dimensions
- discrimination metrics appropriate to each task
- calibration metrics
- robustness / held-out performance
- ablation: structure-informed vs sequence-only
- case-study interpretation of biologically meaningful mutations
