# Interpreting Focus Area 2A for This Project

Focus Area 2A is broad enough to support a virus-centered project if the application remains anchored to biomolecular science.

## Strong 2A alignment signals
- protein structure-function relationships are central
- mutational effects are interpreted through biomolecular context
- structural characterization and omics-derived variation are integrated
- the output advances functional analysis of a biological system

## Weak alignment signals
- purely epidemiologic forecasting
- purely sequence-based label prediction with no structure-function interpretation
- broad biodefense claims without a biomolecular core

## Best framing for this H5N1 proposal
Frame H5N1 as a model system for discovering how viral mutations reshape biomolecular behavior relevant to antigenicity, immune recognition, and host adaptation.
