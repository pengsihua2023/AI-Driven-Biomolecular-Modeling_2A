# Appendix Package Templates for DOE Genesis Mission Phase I (Topic 2A)

This package contains editable markdown templates for the Project Narrative appendices and related supporting materials.

## Recommended use

1. Keep `proposal/03_project_narrative.md` as the main narrative.
2. Draft each appendix in the matching template file.
3. Convert the final narrative and appendices into **one Project Narrative PDF** for submission.
4. Do **not** upload separate appendix files to Fields 9, 10, 11, or 12 on the Research and Related Other Project Information form.
5. Keep all Letters of Commitment and letters of collaboration/access in **Appendix 7**.
6. Treat the DMSP template as a preparatory file for **award negotiation**, not as a required narrative appendix.

## Editing principle

Use plain, specific, auditable language. Avoid marketing phrases, recommendation-style letters, and anything that looks like page-limit circumvention.
