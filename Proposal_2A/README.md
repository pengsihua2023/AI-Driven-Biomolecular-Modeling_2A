# DOE 2A Proposal Scaffold for Claude Code

This repository is a context-engineered and harness-engineered workspace for drafting a DOE Genesis Mission FY26 Phase I proposal under Focus Area 2A (Biomolecular Science, BER).

Working title:
**AI-Driven Biomolecular Modeling of H5N1 Antigenic Evolution, Immune Escape, and Cross-Species Adaptation**

## Why this scaffold exists
This project is designed so Claude Code can operate with:
- sponsor-aware context
- consistent terminology
- strong proposal structure
- explicit compliance rules
- reviewer-style self-critique
